# Varsaflow Widget

1. Upload widget.js to a GitHub repository.
2. Use jsDelivr CDN:

https://cdn.jsdelivr.net/gh/YOUR_GITHUB_USERNAME/varsaflow-widget@main/widget.js

3. Paste this into Shopify theme.liquid before </body>:

<script src="https://cdn.jsdelivr.net/gh/YOUR_GITHUB_USERNAME/varsaflow-widget@main/widget.js"></script>
